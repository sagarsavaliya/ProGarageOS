Cursor Screen 01 · Splash Screen · Staff App

  • State management: No Riverpod provider needed. Pure StatefulWidget with AnimationController(vsync: this, duration: 3000ms). Auto-navigate via Future.delayed(const Duration(milliseconds: 2500), () => context.go('/staff/login')).

  • Widget tree: Scaffold(backgroundColor: AppColors.slate900) → Stack [ DotGridPainter(CustomPainter), AmbientGlowWidget, SafeArea → StatusRow, Center → WordmarkColumn, Positioned(bottom:74) → ContinueButton, Positioned(bottom:50) → ProgressBar, Positioned(bottom:30) → VersionText, HomeIndicator ].

  • DotGridPainter: CustomPainter override paint(Canvas c, Size s) — nested for loops across x: 0..s.width/28 and y: 0..s.height/28. Each iteration: c.drawCircle(Offset(x*28+14, y*28+14), 1.2, paint) where paint.color = Colors.white.withOpacity(0.028).

  • Animation sequence: _ctrl.forward() drives three intervals: 0–0.23 → wordmark FadeTransition + SlideTransition(Offset(0, 0.04)→zero); 0.25–0.95 → LinearProgressIndicator.value; 0.95–1.0 → CTA AnimatedOpacity. Use CurvedAnimation(parent: _ctrl, curve: Interval(...)) per stage.

  • Navigation: context.go('/staff/login') called from timer OR button tap (first wins). Route defined in AppRouter — /staff/login → StaffLoginScreen(). Transition: CustomTransitionPage with fade + scale(0.97→1).
