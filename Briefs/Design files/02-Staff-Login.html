Cursor Screen 02 · Staff Login · PIN Auth

  • Providers: staffLoginProvider (StaffLoginNotifier) holds StaffLoginState { idle | loading | error | locked }. Methods: addDigit(int), deleteDigit(), clearPin(), submitPin(), triggerBiometric(). savedUserProvider reads last logged-in user from FlutterSecureStorage to pre-fill the user card.

  • PIN logic: submitPin() calls AuthRepository.loginStaff(login, pin) → POST /auth/staff/login. On 401: increment failCount. At failCount ≥ 5: set status = locked, start CountdownTimer(30s). Store failCount + lockExpiry in SharedPreferences so lockout persists across app restarts.

  • Widget tree: PinPadWidget (3×4 grid of PinKeyWidget) + PinDotsWidget (Row of 6 AnimatedContainers). PinKeyWidget: GestureDetector with onTapDown/Up for scale 0.94 animation using AnimatedScale(duration:kInstant, curve:easeIn).

  • Biometric: local_auth package. LocalAuthentication().authenticate(localizedReason:'Sign in to ProGarage'). If authenticated: call AuthRepository.loginStaff with stored credentials. Show fingerprint ripple via AnimatedContainer box-shadow expansion.

  • Navigation: On success: ref.read(authProvider.notifier).saveSession(token, user) → context.go('/staff/dashboard'). Transition: CustomTransitionPage with FadeTransition + ScaleTransition(0.97→1) on enter, FadeTransition on exit (300ms easeExit).
